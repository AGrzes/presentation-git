Installation

  • rpm - yum install git
  • Windows - Download and install
  • Mac - Download and install

Configuration

  • git config --global user.name ${your user name} - set your username for commits
  • git config --global user.email ${your email} - set your email for commits
  • git config --global core.editor ${default editor, like vi, nano ...} - set editor used to edit commit messages
  • git config --global push.default simple - set simple push mode - to prevent git from nagging on every push
  • git config --global credential.helper cache - make git remember your credentials for few minutes instead of asking each time

Clone

  • cd ${working directory} - move to place where you plan to clone repository
  • git clone https://hub.jazz.net/git/AGrzes/git-workshop1 - clone workshop repository into git-workshop1 subdirectory

Basic Excrises

  • Add a file
  • Modify a file
  • Revert a change
  • Revert once again
  • And again

Add a file

  • echo "Hello Git!" > hello.${your name} - create your own file
  • git add hello.${your name} - add file to index
  • git status - check work status
  • git commit -m "My first commit" - commit
  • git status - check work status
  • git log --oneline --graph --decorate - see log

Modify a file

  • echo "Goodbye Git :(" > hello.${your name} - change the file content
  • git diff hello.${your name} - add file to index
  • git add hello.${your name} - add change to index
  • git status - check work status
  • git commit -m "Second commit" - commit
  • git status - check work status
  • git log --oneline --graph --decorate - see log

Revert a change

  • git checkout HEAD~1 hello.${your name} - get the previous version of the file
  • cat hello.${your name} - see the file content
  • git status - check work status
  • git add hello.${your name} - add change to index
  • git commit -m "Reverse second commit" - commit
  • git status - check work status
  • git log --oneline --graph --decorate - see log

Revert once again

  • git revert HEAD - revert the last commit
  • cat hello.${your name} - see the file content
  • git status - check work status
  • git log --oneline --graph --decorate - see log

And again

  • git reset --hard HEAD~1 - rewind history discarding last commit
  • cat hello.${your name} - see the file content
  • git status - check work status
  • git log --oneline --graph --decorate - see log

Synchronize

  • Pull
  • Push
  • Fetch
  • Merge

Pull

  • git pull --rebase - get the changes from remote repository and apply own changes on top of them
  • ll - see pulled changes
  • git log --oneline --graph --decorate - see log

push

  • git push - share the changes with remote repository
  • If failed repeat pull

Fetch

  • git fetch - get the changes from remote repository without merging them with your work
  • git log master origin/master --oneline --graph --decorate - see log of local and remote work

Merge

  • git merge origin/master - apply own changes on top of remote changes

Branching

  • Branch
  • Rebase
  • Rewrite History
  • Merge Changes

Branch

  • git branch fix - create branch fix from master
  • git checkout fix - switch to branch fix
  • echo "${your name}.1" >> shared - change the file content
  • git commit -a -m "Fix 1" - commit
  • echo "${your name}.2" >> shared - change the file content
  • git commit -a -m "Fix 2" - commit
  • echo "${your name}.3" >> shared - change the file content
  • git commit -a -m "Fix 3" - commit
  • git log --oneline --graph --decorate - see log

Rebase

  • git checkout master - switch back to branch master
  • git pull --rebase - get the changes from remote repository and apply own changes on top of them
  • git checkout fix - switch to branch fix
  • git rebase master - apply fix changes on top of master branch
  • ${editor} shared - resolve conflict
  • git add shared - mark as resolved<
  • git rebase --continue - finish rebase
  • cat shared - see resulting file
  • git log --oneline --graph --decorate - see log

Rewrite History

  • git rebase -i master - rewrite branch history from divergence point
  • reword first commit and fixup the next two
  • git commit --amend -m "I did it in one go" - amend commit message
  • git rebase --continue - finish rebase
  • git log --oneline --graph --decorate - see log

Merge Changes

  • git checkout master - switch back to branch master
  • git pull --rebase - get the changes from remote repository and apply own changes on top of them
  • git merge fix - merge fix changes to the master branch
  • ${editor} shared - resolve conflict
  • git add shared - mark as resolved
  • git commit - finish rebase
  • cat shared - see resulting file
  • git push - share the changes with remote repository
  • If failed repeat pull and resolve conflicts
  • git log --oneline --graph --decorate - see log
